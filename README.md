# Taller de Listas Simples en Python

## Descripción del proyecto

Este proyecto consiste en el desarrollo de una aplicación en **Python** que permite gestionar una **lista de tareas pendientes** mediante el uso de una **lista simplemente enlazada**.

Cada nodo de la estructura representa una tarea y contiene una referencia al siguiente nodo, siguiendo el enfoque trabajado en clase para el tema de **listas simples** dentro de la asignatura de **Estructuras de Datos**.

Además, el sistema cuenta con una **interfaz gráfica en español**, desarrollada para facilitar la interacción del usuario de manera visual, clara y ordenada.

---

## Objetivo

Implementar una solución en Python que permita trabajar con una lista de tareas pendientes utilizando una **lista simple**, aplicando conceptos de:

- Estructuras de datos
- Programación orientada a objetos
- Interfaz gráfica de usuario

---

## Características principales

- Registro de tareas pendientes
- Visualización de tareas almacenadas
- Implementación de nodos enlazados
- Uso de programación orientada a objetos
- Interfaz gráfica desarrollada en Python
- Diseño sencillo, académico y funcional

---

## Estructura del proyecto

El proyecto está compuesto por las siguientes clases principales:

### `TaskNode`
Representa cada nodo de la lista simple.  
Cada nodo almacena:

- La tarea
- La referencia al siguiente nodo

### `TaskList`
Administra la lista simplemente enlazada.  
Permite:

- Agregar tareas
- Recorrer la lista
- Mostrar las tareas registradas

### `TaskManagerGUI`
Gestiona la interfaz gráfica del sistema.  
Permite al usuario:

- Ingresar tareas
- Agregar tareas a la lista
- Visualizar las tareas almacenadas
